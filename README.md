## To Start

### 1. Renaming config and message file
> Rename config file `.env.example` to `.env` and `message.example.json` in `src/text` to `message.json`.

### 2. Replace config and message
> Fill `.env` and `message.json`

## Install with shell

### 1. Init Project
```bash
make init
```

### 2. Build Project
```bash
make build
```

### 3. Starting
```bash
make start
```

## Install with Docker

### 1. Build Image
```bash
make docker_build
```

### 2. Start Image
```bash
make docker
```
